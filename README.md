# 자기소개 위키 페이지 만들기
---
레벨1 공통 교육-자기 소개 위키 페이지 만들기

- 깃 초기화 및 HTML, CSS, JS, 이미지 리소스 디렉토리 계층 분리
- HTML DOM 트리 아키텍처 작성
  - `index.html` 에 `<header>`, `<main>`, `<aside>` 등의 태그로 페이지의 뼈대 구성
  - 텍스트, 이미지 등 정적 데이터 주입
- CSS 파일 작성
  - `reset.css` 을 통한 브라우저 기본 스타일 무효화
  - `wiki-layout.css` 을 통한 화면 분할, 2차원 레이아웃 배치
  - `common.css`, `main.css` 로 세부 속성 설정
- 자바스크립트 작성을 통한 댓글 작성 기능 구현
  - 브라우저 기본 동작(폼 전송 후 자동 새로고침)을 제어
  - 메모리 상의 DOM 트리를 동적으로 조작하여 댓글 작성 기능 구현
- GitHub Pages 를 통한 정적 웹 페이지 배포 및 지속적 배포 테스트
